# Security Policy

## 🛡️ Our Commitment

We take the security of OCR Agent very seriously. Your security and privacy are our top priorities. If you find any security vulnerabilities, please let us know immediately through the contact methods below.

## 📝 Reporting a Vulnerability

Please do **not** file public issues for security vulnerabilities. If you discover a potential vulnerability, please report it privately via email to:

**[shibilahamed7012@gmail.com](mailto:shibilahamed7012@gmail.com)**

We will process your report and provide an initial response within 48-72 hours. We will follow up with more information and coordinate a timeline for disclosure at our discretion.

## 🕰️ Vulnerability Response Process

1. **Acknowledgment**: We will acknowledge your report and begin an investigation.
2. **Investigation**: We will work to verify and understand the scope of the vulnerability.
3. **Mitigation**: Once verified, we will work on a fix as quickly as possible.
4. **Resolution**: We will deploy the fix and notify you of the resolution.

## 🔒 Confidentiality

We value the contributions of the security research community. All reports will be handled with strict confidentiality.

## 📢 Public Disclosure

We ask that you do not disclose security vulnerabilities publicly until we have had a chance to fix them. We will coordinate any public disclosure (if necessary) with you as the reporter.

---

Thank you for helping keep OCR Agent secure! 🚀
