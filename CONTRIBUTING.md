# Contributing to OCR Agent (LeaseGuardAI)

Thank you for your interest in contributing! We welcome contributions that help make OCR Agent the best tool for analyzing lease contracts.

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## 🚀 How to Get Started

### 1. Fork the Repository
First, create your own fork of the repository and clone it to your local machine.

### 2. Set Up the Local Environment
Follow the **Quick Start** instructions in the [README.md](README.md) to set up:
- Infrastructure via Docker
- Backend (Node.js/Express)
- Frontend (React/Vite)

### 3. Work on Your Changes
Create a descriptive feature branch:
```bash
git checkout -b feat/your-feature-name
```
Or for bug fixes:
```bash
git checkout -b fix/your-bug-description
```

### 4. Code Standards
- **TypeScript**: All new code should be written in TypeScript with proper typing.
- **Linting**: Ensure your code follows the project's formatting and linting rules.
- **Testing**: If possible, add tests for your changes.

### 5. Commit and Push
Ensure your commit messages are clear and follow the Conventional Commits format if possible (e.g., `feat: Add more OCR engines`, `fix: Correct residual value calculation`).

### 6. Create a Pull Request
Submit your pull request (PR) against the `main` branch. Please provide a clear description of the changes in the PR template.

## 🐛 Reporting Bugs
If you find a bug, please open an issue using the **Bug Report** template. Provide as much detail as possible, including:
- Steps to reproduce
- Expected vs. actual behavior
- Browser/OS details
- Screenshots (if applicable)

## ✨ Suggesting Enhancements
We love new ideas! Please open an issue using the **Feature Request** template to discuss your ideas before starting implementation.

## 🛡️ Security Vulnerabilities
If you discover a security vulnerability, please follow the instructions in our [SECURITY.md](SECURITY.md) to report it privately.

## 📞 Contact
For any other questions, feel free to open a discussion or reach out to the project maintainers.

---

Happy contributing! 🚀
